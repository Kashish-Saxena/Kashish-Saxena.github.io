Projects

Projects

Below are the some of my Data Analytics projects using SQL, Python, Tableau & Excel.

Bank Customer Churn Analysis & Prediction

This analysis project investigates and gains insights into the major factors impacting the bank's customer churn rate using Python. Subsequently, the study aims to provide recommendations to the bank, thereby facilitating a reduction in churn rate and improving customer retention. In addition, a machine learning model is deployed to accurately predict customer churn.

KPMG Data Analytics Virtual Internship Experience

Sprocket Central Pty Ltd has given KPMG a list of 1000 potential customers with their demographics and attributes. However, these customers do not have a prior transaction history with the organization. The objective of this project is to analyze the existing customer and transactions data to reveal useful customer insights that could help optimize resource allocation for targeted marketing by selecting top target customers from the list of potential customers.

Customer Sales Analytics Dashboard using Tableau

This project visually analyzes a customer data using Tableau. The following key metrics are analyzed: Total revenue per state, Total revenue per category by gender, Total sales by category, Percent revenue by region, Total sales by age, Total revenue by gender, Total revenue per month and the Discount-Quantity correlation.


World Happiness Report Exploratory Data Analysis

This project performs Exploratory Data Analysis (EDA) using Python on the happiness rate of countries based on attributes such as happiness score, social support, GDP per capita, family, life expectancy, perceptions of corruption, etc. and analyzes the effect of COVID-19 on the individual attributes as well as the overall happiness of countries.

Nashville Housing Data Cleaning and Analysis using SQL

This project involves the cleaning and analysis of a Nashville housing dataset using SQL to derive insights for stakeholders in the real estate domain. The dataset, consisting of 56,477 records, encompasses various attributes such as land use, sale prices, property details, and ownership information.

OfficeStaples Inc. Sales Analysis

The objective of this project is to examine the sales data of OfficeStaples Inc. using Excel and develop effective strategies for boosting sales. The store also intends to introduce a new product category, printers and has set a specific sales target for the upcoming year. To meet this objective, an additional goal is to estimate the expected total revenue generated by the introduction of this new printer category.
